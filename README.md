План для разработки системы управления фитнес-клубом «FitZone»
1 неделя:
Постановка задачи и предметной области, входные/выходные данные, выписать список всех сущностей и атрибутов, черновой словарь данных, каркас репозитория гит, readme файл с планом. 
2 неделя:
Создание ERD (датологической) модели, нормализация до 3 уровня, план индексов, словарь данных, дополнение пояснительной записки, план миграции первая версия. 
3 неделя:
физическая база данных, набор тестовых данных, базовые круд-операции и view, структурная схема, функциональная схема, обновление пояснительной записки.
4 неделя:
 Серверная логика, процедуры, триггеры и транзакции с обработкой ошибок, полный пакет sql-запросов, описание всех контрактов и кодов ошибок для api, обновление пояснительной записки.
5 неделя:
Безопасность и администрирование, разграничение прав доступа, хэширование, логирование пользовательских действий, скрипт и бэкапы, реализация маск для полей.
6 неделя:
Разработка api и клиентского приложения, аутентификация, роли, адаптивность, импорт-экспорт данных (ф форматах csv, sql, json..). схема интерфейса и руководство пользователя. 
7 неделя:
Аналитика и тестирование, полный цикл тестирования. 
8 неделя:
Итог и полный рабочий проект, презентация.
